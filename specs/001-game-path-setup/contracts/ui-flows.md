# Contracts: UI Flows (Game Path Setup)

Created: 2026-01-22

## Settings

- User can set, change, and clear the Game Installation Directory.
- User can manage Scan Directories independently.

## Local Mod

- If Game Installation Directory is missing: show a clear prompt and a direct way to open Settings.
- If configured: allow normal Local Mod workflows.

## Dashboard

- If Game Installation Directory is missing: show a light onboarding call-to-action to configure it.
- If configured: onboarding prompt hidden.
