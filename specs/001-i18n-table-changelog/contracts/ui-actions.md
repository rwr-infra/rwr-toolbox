# Contract: User-Facing Actions (UI)

本特性不引入对外 API；这里用“用户动作契约”代替接口契约，用于驱动任务拆分与验收。

## Dashboard

- 用户打开首页：所有可见文案必须已国际化（无 raw key）。
- 用户查看“最近更新”：可看到最近 N 条更新摘要；无数据时显示空态。

## About

- 用户打开 About：可以查看完整更新日志；无数据/解析失败时显示空态或回退展示。

## Data Tables (Items / Weapons)

- 用户滚动表格：表头持续可见且稳定；快速/慢速滚动一致。
- 用户切换显示列：表头/表体列对齐不抖动；滚动行为不退化。
